# FreshMart Grocery Chatbot Demo

A ready-to-upload GitHub Pages chatbot project for a small local grocery store.

## Included
- Polished HTML
- Responsive CSS
- JavaScript FAQ chatbot
- Grocery branding
- Bilingual option: English and Somali
- Live demo question buttons

## Files
- `index.html`
- `styles.css`
- `script.js`

## How to use on GitHub Pages
1. Create a new GitHub repository.
2. Upload all project files to the repository root.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` and `/root`
5. Save and wait for GitHub Pages to publish.

## Customize for a real store
Update these in `script.js` and `index.html`:
- Store name
- Address
- Phone number
- Hours
- Product categories
- Delivery rules
- Language text
